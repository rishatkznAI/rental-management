# Release Notes

## Обновление авторизации

Вход теперь выполняется по логину — части технического email до символа @.
Например, пользователь с техническим email ivanov@skytech.ru может входить как ivanov.

Полный email по-прежнему можно вводить для обратной совместимости.
Пароли, роли, права доступа и существующие сессии не изменились.

MAX-бот теперь также просит логин вместо email.
