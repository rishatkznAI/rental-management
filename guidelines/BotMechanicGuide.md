# Сценарий работы механика с ботом

Этот сценарий нужен для механиков, которые работают с сервисными заявками через бот в MAX.

## Что бот умеет

Через бота механик может:

- войти в систему под своим логином и паролем
- посмотреть свои заявки
- взять заявку в работу
- добавить работы по заявке
- добавить запчасти
- сохранить черновик результата
- перевести заявку в статус `Готово`
- закрыть заявку по чек-листу

## Как начать работу

1. Откройте бот.
2. Нажмите `Войти`.
3. Введите свой `email`.
4. Следующим сообщением введите пароль.

После входа бот покажет доступные действия.

## Основные команды

- `/моизаявки` — мои активные заявки
- `/сервис` — список сервисных заявок
- `/вработу ID` — взять заявку в работу
- `/ремонт ID` — открыть заявку и работать по ней
- `/работы` — начать поиск и добавление работ
- `/запчасти` — начать поиск и добавление запчастей
- `/черновик` — посмотреть текущий черновик по заявке
- `/итог` — сохранить итог ремонта текстом
- `/готово` — отметить, что работы завершены
- `/закрыть` — закрыть заявку
- `/сброс` — сбросить текущий пошаговый сценарий, если бот “застрял”

## Базовый рабочий сценарий

### 1. Найти свою заявку

Напишите:

```text
/моизаявки
```

Бот покажет ваши заявки.

### 2. Взять заявку в работу

Напишите:

```text
/вработу S-123456
```

Или откройте заявку кнопкой из списка.

После этого бот переведёт заявку в работу и откроет рабочее меню по ремонту.

### 3. Добавить работы

Напишите:

```text
/работы
```

После этого бот попросит ввести текст для поиска.

Примеры:

```text
гидравлика
замена масла
диагностика
```

Что происходит дальше:

1. Механик пишет текст.
2. Бот ищет совпадения в справочнике работ.
3. Бот показывает найденные варианты кнопками.
4. Механик нажимает нужную работу.
5. Бот просит указать количество.
6. Механик отправляет число, например:

```text
2
```

После этого работа добавляется в заявку.

### 4. Если нужная работа не нашлась

Просто напишите другой поисковый запрос.

Например:

```text
насос
редуктор
проводка
```

Бот заново выполнит поиск и предложит новый список.

### 5. Добавить запчасти

Напишите:

```text
/запчасти
```

Дальше логика такая же:

1. введите текст для поиска
2. бот покажет найденные запчасти
3. выберите нужную кнопку
4. укажите количество
5. при необходимости укажите цену

Пример поиска:

```text
фильтр
масло
насос
```

### 6. Посмотреть черновик

Напишите:

```text
/черновик
```

Бот покажет, что уже добавлено:

- работы
- запчасти
- итог
- статус по заявке

### 7. Сохранить итог ремонта

Напишите:

```text
/итог
```

После этого следующим сообщением отправьте текстом результат ремонта.

Пример:

```text
Заменили гидрошланг, проверили давление, течь устранена.
```

### 8. Отметить, что работа закончена

Когда ремонт завершён, напишите:

```text
/готово
```

Это переводит заявку в статус `Готово`.

### 9. Закрыть заявку

Когда всё полностью завершено, напишите:

```text
/закрыть
```

Бот проведёт по чек-листу закрытия и затем закроет заявку.

## Рекомендуемый рабочий порядок

Для механика лучше работать в таком порядке:

1. `/моизаявки`
2. `/вработу ID`
3. `/работы`
4. `/запчасти`
5. `/черновик`
6. `/итог`
7. `/готово`
8. `/закрыть`

## Как правильно искать работы

Лучше писать короткие и понятные запросы:

- `гидравлика`
- `масло`
- `диагностика`
- `редуктор`
- `электрика`
- `насос`

Лучше не писать слишком длинную фразу целиком.

Неудачный пример:

```text
нужно поменять шланг гидравлики на стреле
```

Лучше так:

```text
гидравлика
шланг
```

## Если бот не понял, что делать

Если бот “застрял” в старом шаге или вы ошиблись, напишите:

```text
/сброс
```

После этого снова откройте нужную заявку:

```text
/ремонт ID
```

## Короткий пример полного диалога

```text
/моизаявки
/вработу S-123456
/работы
гидравлика
<нажать нужную работу>
2
/запчасти
фильтр
<нажать нужную запчасть>
1
/черновик
/итог
Заменили фильтр, устранили течь, проверка выполнена.
/готово
/закрыть
```

## Важное правило

Сначала механик фиксирует:

- какие работы сделаны
- какие запчасти использованы
- итог ремонта

И только потом переводит заявку в `Готово` и закрывает её.
